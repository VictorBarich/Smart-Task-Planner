Current model: Gemini 2.5 Flash

In order to run this:

1. Make your API key at https://aistudio.google.com/api-keys
2. Make an env file
3. Paste your API key into it like this: API_KEY=mykey
4. Run python main.py

Also not sure what the limits are like for the free tier, so might get weird errors if you run it excessively.


The model's reasoning will also acquired in a second call and be printed for manual inspection. This can be commented out if rate limits become a problem.
